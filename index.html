About

Handmade

Live Github

Handmade is a clone of Etsy.com. Users can create an account, buy, and sell products. Handmade was created with a Postgresql and Ruby on Rails backend, and a React.js/Redux.js frontend. Key features include carts and product listing forms. The cart API utilizes a couple joins to bring information from users, products, and cart items to the store and into a normalized state.

Mass

Live Github

Mass is a clone of the game Agar.io where its eat or be eaten! Mass was built with JavaScript and HTML Canvas. It features a camera object that tracks and focuses on the player as they navigate a board much larger than the display and an AI using environmental information to make real time decisions.
